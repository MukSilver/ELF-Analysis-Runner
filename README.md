# Linux ELF Analysis and Restricted Execution Tool

This project aims to analyze security properties of Linux ELF binaries,
execute them in a controlled environment,
and store the results as structured reports.

## 프로젝트 소개
이 프로젝트는 Linux ELF 바이너리의 보안 속성을 분석하고,
제한된 환경에서 실행한 뒤,
그 결과를 구조화된 리포트로 저장하는 도구를 목표로 한다.
